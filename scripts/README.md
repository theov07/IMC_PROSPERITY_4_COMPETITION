# Scripts Layout

- `shared/`: generic utilities used across the framework
- `round_1/` to `round_5/`: research, diagnostics, and one-off tooling specific to each competition round

Typical entry points:

- `scripts/shared/export_submission.py`
- `scripts/shared/analyze_log.py`
- `scripts/shared/analyze_backtest.py`
- `scripts/shared/benchmark_strategy.py`
