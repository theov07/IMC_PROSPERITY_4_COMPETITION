# Old

Ce dossier contient les anciens fichiers archivés après la mise en place du nouveau framework.

## Contenu

- `legacy_docs/`
  Ancienne doc de backtest et fichier de test vide.
- `legacy_strategies/`
  Anciennes stratégies standalone et anciens baselines, remplacés par `submissions/` et le noyau `prosperity/`.
- `legacy_templates/`
  Ancien template `Prerequiste.py`, utile comme référence historique mais plus utilisé dans le workflow actuel.

## Principe

On archive ici ce qui n'est plus dans le chemin principal de travail, sans le supprimer définitivement.
